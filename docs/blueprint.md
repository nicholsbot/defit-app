# **App Name**: DEFIT 2026

## Core Features:

- User Registration and Login: Allow military servicemembers to register and log in securely via the dashboard.
- Dashboard Navigation: Provide easy navigation to Resources, Rules, Leaderboards, Swag, and Learn More sections through header or navigation tabs.
- Fitness Challenge Tracking: Track users' progress in the fitness challenge using measurable metrics.
- Leaderboard Ranking: Display real-time leaderboard rankings to foster competition and motivation.
- Resource Access: Provide access to resources, rules and all other relevant data.
- Workout Logging: Main button to log workouts, directing users to a page that tracks weekly progress in TMAR-M, Strength and Resistance, Cardio, and HIIT categories.
- Progress Tracking: Track weekly progress in each category, visualize completion percentage, and calculate percentages automatically. The challenge is 10 weeks long beginning on 12 Jan, 2026.
- Minimum Threshold Display: Display whether the user has met the 100% minimum for each category.
- TMAR-M Logging: Log TMAR-M workouts, with 60 minutes equaling 100%.
- Resistance Logging: Log Strength and Resistance workouts by lbs lifted. Minimum per week is 5,000lbs lifted. 5,000 = 100%
- Cardio Logging: Log cardio workouts, where running, rucking and walking must be 12 miles per week, Elliptical and rowing is 15 miles per week, Swimming is 3,200 meters, and cycling is 40 miles per week for 100%. The app will calculate the totals.
- HIIT Logging: Log HIIT workouts, based on 45min per week minimum. 45 min = 100%.
- Edit Functionality: Users must be able to edit their totals if they make a mistake.
- Combined Percentage Calculation: Ensure that separate events can be combined together and make up the total percentages.
- Unlimited Logging: Once weekly minimums (100%) are met users can still log; there is no upper limit on how much they can log.

## Style Guidelines:

- Primary color: Army yellow (#F0A30A) to evoke military association.
- Background color: Dark charcoal (#222222), desaturated and dark for a professional feel.
- Accent color: Dark green (#347C4A), analogous to yellow, but different enough to stand out.
- Font: 'PT Sans', a sans-serif font for clear readability in headers and body text.
- Dashboard layout with 'Register Now' and 'Login' buttons side-by-side. Tabbed navigation for primary sections.
- Use military-themed icons where applicable to enhance the visual appeal.
- Subtle animations for user interactions to improve engagement.
